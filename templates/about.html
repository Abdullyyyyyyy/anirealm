About AniRealm

AniRealm is a full-stack web application built with Flask, SQLite, and JavaScript. It allows users to create anime episode journal entries and automatically analyses the emotional tone of their reflections using three different NLP models.

Sentiment Models

📏

VADER

A lexicon and rule-based model designed for social media text. Fast and reliable for short reflections.

📝

TextBlob

A simple NLP library that calculates polarity and subjectivity scores from text.

🧠

DistilBERT

A transformer-based deep learning model fine-tuned on SST-2. Understands context better than rule-based methods.

Evaluation Metrics

🎯

Accuracy

The proportion of correct predictions out of all predictions made.

🔬

Precision

How many of the predicted positives were actually correct.

📡

Recall

How many actual positives were correctly identified by the model.

⚖️

F1-Score

The balanced average of precision and recall into one score.

Technology Stack

🐍

Python & Flask

Backend framework handling routes, API logic, and authentication.

🗄️

SQLite & SQLAlchemy

Database storage with ORM model definitions.

🎨

HTML, CSS & JS

Frontend interface with dynamic rendering and Fetch API communication.

🔒

Flask-Login & Werkzeug

Secure user authentication with password hashing and session management.
